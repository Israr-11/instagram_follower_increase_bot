Getting Started (In Terminal)

  1. First Time Setup:
    • Run python interactive_bot.py in terminal
    • Wait for bot to initialize and login
    • You'll see an interactive menu with options
    • Choose option 5 to adjust settings
  2. Configure Settings:
    • Set follower range (500-50,000 recommended)
    • Adjust delays (30-60 seconds for safety)
    • Enable/disable filters (tech-only, bio check, etc.)
    • Set max follows per hashtag
  3. Search by Hashtags (Recommended):
    • Choose option 1 (custom hashtag) or 2 (predefined)
    • Enter hashtag without # symbol
    • Set how many users to follow
    • Watch Chrome browser work automatically
  4. Follow Specific Users:
    • Choose option 3 from menu
    • Enter usernames separated by commas
    • Bot will check criteria and follow
  5. Monitor Progress:
    • Choose option 6 to see statistics
    • Terminal shows real-time activity logs
    • Check followed_users.json for details

‚ö†Ô∏è Important Tips

Stay Safe:
  • Start slow: Follow 10-20 users per day initially
  • Gradually increase to 30-50 per day maximum
  • Keep delays between 30-60 seconds
  • Don't run the bot 24/7 - act human!
  • Use tech-only filter to target relevant accounts

üéØ Best Practices

  • Hashtag Strategy: Use specific hashtags like #100daysofcode, #learntocode
  • Follower Range: Target 500-10,000 followers for best follow-back rates
  • Tech Filter: Enable if you want coding/tech audience only
  • Timing: Run during peak hours (9am-5pm in your timezone)
  • Follow-backs: Check after 3-7 days, unfollow non-followers

‚ùì Troubleshooting

  • Bot won't start: Make sure Chrome is installed and Python is in PATH
  • Login fails: Check username/password in config.py
  • No follows happening: Lower max followers or disable tech filter
  • Too many skips: Adjust follower range to be more inclusive
  • Browser closes: Don't close it manually, use terminal controls
